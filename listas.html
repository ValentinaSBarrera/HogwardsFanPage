Hechizos

Tipos de Hechizos

Transformaciones
- También conocidos como hechizos de transfiguración, alteran la forma o apariencia del objetivo.
  • Transformación
    • Draconifors
  • Destransformación
    • Reparifarge
  • Permutacion
    • Felifors
  • Desvanecimiento
    • Evanesco
  • Conjuración
    • Aguamenti
Encantamientos
- Estos hechizos alteran las propiedades de sus objetivos, como sus comportamientos y propiedades.
  • Nox
  • Accio
  • Engorgio
  • Ascendio
  • Desmaius
  • Riddikulus
  • Colloportus
  • Expelliarmus
  • Protego Totalum
  • Expecto Patronum
  • Arresto Momentum
  • Wingardium Leviosa
Embrujos
- Son hechizos cuyos efectos son irritantes, pero divertidos, y que generan inconvenientes menores al objetivo, tienen una ligera connotación de magia oscura.
  • Ebublio
  • Relashio
  • Flipendo
  • Oppugno
  • LeviCorpus
  • Impedimenta
Maleficio
- Afectan al objetivo de manera negativa; tiene una connotación de magia oscura, ligeramente peor que un embrujo. Genera grandes inconvenientes a la víctima.
  • Titillando
  • Densaugeo
  • Anteoculatia
  • Entomorphis
  • Mutatio Skullus
  • Engorgio Skullus
Maldiciones
- Se reservan para los peores tipos de magia oscura, con la intención de afectar al objetivo de manera sumamente negativa.
  • Crucio
  • Expulso
  • Imperio
  • Reducto
  • Confringo
  • Avada Kedavra
  • Sectumsempra
  • Petrificus Totalus
Contrahechizos
- Inhibición o finalización del efecto de otro hechizo.
  • Lumus
  • Quietus
  • Evanesco
  • Alohomora
  • Vulnera sanentur
Hechizos de Curación
- Mejora la condición de los seres vivos.
  • Episkey
  • Anapneo
  • Reparifors
  • Vulnera Sanentur

Tutorial de como hacer listas y sus tipos

Para hacer una lista podemos elegir 3 diferentes etiquetas ul para listas desordenadas, ol para listas ordenadas y dl para listas de definición
Para agregar un elemento a la lista usamos la etiqueta li y para las listas de definición usamos las etiquetas dt para el termino y dd para las definiciones
